# Rocm Model Optimizer

Model optimization toolkit for AMD ROCm

## Features

- Optimized for AMD Instinct MI300X accelerators
- ROCm 6.0+ compatible
- Python 3.8+ with PyTorch ROCm support
- JSON output for CI integration

## Requirements

- AMD GPU with ROCm 6.0+
- Python 3.8+
- PyTorch with ROCm support

## Quick Start

```bash
git clone https://github.com/blinksz/rocm-model-optimizer.git
cd rocm-model-optimizer
pip install -r requirements.txt
python main.py --benchmark
```
